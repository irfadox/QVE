# QVE
This repository contains a research codebase for evaluating the **Variational Quantum Eigensolver (VQE)** for molecular simulations (H₂, LiH) under NISQ constraints.   The project investigates **Layer-wise Unitary Folding** for Zero-Noise Extrapolation (ZNE), optimizer resilience, and the impact of topological depth on decoherence.
